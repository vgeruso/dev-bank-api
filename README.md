# dev-bank
### API of a virtual bank

![mainImage](https://i0.wp.com/www.thestartupfounder.com/wp-content/uploads/2019/01/digital_banking.jpg?fit=608%2C336&ssl=1)

To start your API dev-bank:

cd dev-bank

Create the file .dotenv with the content of .env.example and set variable 'PORT'

Run Your project:
yarn dev
or
yarn start

will be running on: http://localhost:PORT/api/
